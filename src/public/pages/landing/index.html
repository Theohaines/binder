Hosting a game

If you want to host a game create a lobby using the create lobby button!

You can share this 'code' with the other people you're playing with so they can submit their profiles to your lobby!

Joining a game

If you're joining a game you can will need a lobby code from the person hosting your game!

Once you have this lobby code just paste it into the create profile lobby code box and then press create profile!

Viewing a lobby

Once all players have submitted their profiles you can view the lobby!

Just paste the lobby code into the view lobby code box and press the view lobby button!

This will give you a list of all profiles created in your lobby which you can view and vote on which ones the best!

Rules

There is no time limit on games, you can spend as long as you like creating your profile although you may choose to impose a time limit if you'd like!

All information on the profile must be filled out

Public

You can also choose to not join a lobby and just post a profile to the public lobby board!

You can do this by not entering a lobby code and just pressing create profile

Then if you want to see other public profiles just type 'public' into the view lobby box!
